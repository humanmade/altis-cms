# Block Editor

The block editor is the core editing experience of the CMS and provides a true WYSIWYG experience.

## Synced Patterns

WordPress provides built-in support for Synced Patterns (formerly known as Reusable Blocks). These allow you to create blocks of content that can be reused across multiple posts or pages. Any updates to a synced pattern will be reflected everywhere it is used.

Synced patterns can be managed via **Appearance > Editor > Patterns** in the admin, or directly from the block inserter's **Synced** tab.
